Object Oriented Programming -- methodology.
1. class -- blueprint of prgm. Defines the structure.
2. object -- instance of a class. Creates space in our memory.
3. inheritance -- accessing one class properties as another class properties.
4. abstraction -- hiding the unneccesary info.
5. Encapuslation -- making the data accessable only within the class.

Rules : 
1. We always write the code in classes
2. we always use objects to store/ access the data.
